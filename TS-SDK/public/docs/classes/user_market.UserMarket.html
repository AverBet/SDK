    Returns

    TransactionInstruction object

    Parameters

    • outcomeIndex: number

      ID of outcome

    • side: Side

      Side object (bid or ask)

    • limitPrice: number

      Limit price - in probability format i.e. in the range (0, 1). If you are using Decimal or other odds formats you will need to convert these prior to passing as an argument

    • size: number

      Size - in the format specified in size_format. This value is in number of 'tokens' - i.e. 20.45 => 20.45 USDC, the SDK handles the conversion to u64 token units (e.g. to 20,450,000 as USDC is a 6 decimal place token)

    • sizeFormat: SizeFormat

      SizeFormat object (Stake or Payout)

    • orderType: OrderType = OrderType.Limit

      OrderType object. Defaults to OrderType.LIMIT.

    • selfTradeBehavior: SelfTradeBehavior = SelfTradeBehavior.CancelProvide

      Behavior when a user's trade is matched with themselves. Defaults to SelfTradeBehavior.CANCEL_PROVIDE.

    • averPreFlightCheck: boolean = false

      Clientside check if order will success or fail. Defaults to False.

    Returns Promise<TransactionInstruction>

      Owner of UserMarket account. Pays transaction fees.

    • outcomeIndex: number

      Index of the outcome intended to be traded

    • side: Side

      Side object (bid/back/buy or ask/lay/sell)

    • limitPrice: number

      Limit price - in probability format i.e. in the range (0, 1). If you are using Decimal or other odds formats you will need to convert these prior to passing as an argument

    • size: number

      Size - in the format specified in size_format. This value is in number of 'tokens' - i.e. 20.45 => 20.45 USDC, the SDK handles the conversion to u64 token units (e.g. to 20,450,000 as USDC is a 6 decimal place token)

    • sizeFormat: SizeFormat

      SizeFormat object (Stake or Payout formats supported)

    • Optional sendOptions: SendOptions

      Options to specify when broadcasting a transaction

    • Optional manualMaxRetry: number

      No. of times to retry in case of failure

    • orderType: OrderType = OrderType.Limit

      OrderType object. Defaults to OrderType.LIMIT. Other options include OrderType.IOC, OrderType.KILL_OR_FILL, OrderType.POST_ONLY.

    • selfTradeBehavior: SelfTradeBehavior = SelfTradeBehavior.CancelProvide

      Behavior when a user's trade is matched with themselves. Defaults to SelfTradeBehavior.CANCEL_PROVIDE. Other options include SelfTradeBehavior.DECREMENT_TAKE and SelfTradeBehavior.ABORT_TRANSACTION

    • averPreFlightCheck: boolean = true

      Clientside check if order will success or fail. Defaults to True.
