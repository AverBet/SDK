Static Private invertPrice

  • invertPrice(price: Price, uiAmount?: boolean): Price

  • Inverts prices

    This is used when inverting the second outcome in a two-outcome market. When switching prices between bids and asks, the price is 1-p. Example, a BUY on A at a (probability) price of 0.4 is equivelant to a SELL on B at a price of 0.6 (1-0.4) and vice versa.

    Returns

    • Inverted Price object

    Parameters

    • price: Price

      Price object

    • Optional uiAmount: boolean

      Converts prices based on decimal precision if true. Defaults to False.

    Returns Price
